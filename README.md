# student_score_predictor_tool

## Problem
Students dont know that how much hours should we study to get how much marks.

## Solution
my ML model predict marks according to how much hours you study.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib

## How to Run
1. Open file in Google Colab
2. Run all cells
3. Enter study hours to get predicted marks

## Result
- Algorithm: Linear Regression
- Accuracy: 96%
